# WooCommerce SEO Filter — Документация для Claude AI

**Версия:** 1.2.4 | **Статус:** Готово к использованию | **Обновлено:** 2026-03-27

Плагин реализует SEO-оптимизированную фильтрацию товаров через стандартные категории WooCommerce вместо query string параметров.
- Плохо: `site.ru/category/?filter_attr=value`
- Хорошо: `site.ru/category/subcategory/`
