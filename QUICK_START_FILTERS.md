# Быстрый старт: Вывод фильтров

## ✨ НОВОЕ: Универсальный виджет фильтров

Теперь не нужно добавлять отдельный виджет для каждого атрибута!

### Самый простой способ (РЕКОМЕНДУЕТСЯ):

1. Перейдите в **SEO Filter → Filter Attributes**
2. Нажмите **"Добавить категорию"** и выберите нужную категорию
3. Отметьте галочками атрибуты, которые будут доступны для фильтрации
4. Настройте режим отображения (фильтры/подкатегории/оба)
5. Нажмите **"Сохранить все настройки"**
6. Перейдите в **Внешний вид → Виджеты**
7. Добавьте виджет **"SEO Universal Filters"**
8. Готово! Виджет автоматически покажет все фильтры для каждой категории

---

## Преимущества универсального виджета

✅ **Один виджет для всех категорий** - не нужно дублировать  
✅ **Автоматическое определение атрибутов** - показывает только настроенные атрибуты  
✅ **Работает везде** - на SEO-категориях, обычных категориях, даже на главной магазина  
✅ **Легко управлять** - все настройки в одном месте (Filter Attributes)  

---

## Настройка фильтров для категорий

### Шаг 1: Добавление категорий

1. **SEO Filter → Filter Attributes**
2. В разделе "Добавить категорию":
   - Выберите категорию из списка
   - Нажмите "Добавить категорию"
3. Категория появится в списке ниже

### Шаг 2: Настройка атрибутов

Для каждой добавленной категории:

**Доступные атрибуты для фильтра:**
- Отметьте галочками атрибуты (материал, тип, назначение и т.д.)
- Эти атрибуты будут показаны в фильтре на странице категории

**Источник атрибутов:**
- *Из родительской категории* - использовать атрибуты родителя (если дочерняя не настроена)
- *Из своей категории* - использовать только свои атрибуты

**Режим отображения:**
- *Показывать только фильтры* - только фильтры
- *Показывать только подкатегории* - только подкатегории
- *Показывать и фильтры, и подкатегории* - всё вместе

### Шаг 3: Сохранение

Нажмите **"Сохранить все настройки"**

## Как вывести универсальный фильтр

### 1. Через виджеты WordPress (РЕКОМЕНДУЕТСЯ)

1. **Внешний вид → Виджеты**
2. Найдите **"SEO Universal Filters"**
3. Перетащите в сайдбар
4. Настройте:
   - Заголовок: "Фильтры" (или оставьте пустым)
   - Тип отображения: Список или Выпадающий список
5. Сохраните

**Готово!** Виджет будет показывать разные фильтры на разных категориях.

### 2. В functions.php темы

```php
add_action('woocommerce_sidebar', 'add_universal_filters', 5);

function add_universal_filters() {
    if (!is_shop() && !is_product_category()) {
        return;
    }
    
    the_widget('WSF_Universal_Filters_Widget', [
        'title' => 'Фильтры',
        'display_type' => 'list'
    ]);
    
    // Активные фильтры
    the_widget('WSF_Active_Filters_Widget', [
        'title' => 'Выбранные фильтры'
    ]);
}
```

### 3. В шаблоне темы

```php
<?php
// В sidebar-shop.php или woocommerce/archive-product.php

if (is_shop() || is_product_category()) {
    ?>
    <div class="shop-filters">
        <?php
        the_widget('WSF_Universal_Filters_Widget', [
            'title' => 'Фильтры',
            'display_type' => 'list'
        ]);
        
        the_widget('WSF_Active_Filters_Widget');
        ?>
    </div>
    <?php
}
?>
```

## Старый способ (отдельные виджеты)

Если нужно показать конкретный атрибут:

### Виджет "SEO Filter Widget"

1. **Внешний вид → Виджеты**
2. Найдите **"SEO Filter Widget"**
3. Настройте:
   - Заголовок: "Материал"
   - Атрибут: `material`
   - Тип: Список
4. Повторите для других атрибутов

## Примеры настройки

### Пример 1: Магазин с резиновыми покрытиями

**Категории:**
- Резиновые покрытия
- Резиновая плитка
- Резиновые рулоны

**Настройка в Filter Attributes:**

```
Резиновые покрытия:
✓ material (Материал)
✓ tip (Тип)
✓ naznachenie (Назначение)

Резиновая плитка:
✓ naznachenie (Назначение)
✓ razmer (Размер)
✓ tolshchina (Толщина)

Резиновые рулоны:
✓ naznachenie (Назначение)
✓ shirina (Ширина)
```

**Результат:**
- На странице "Резиновые покрытия" - 3 фильтра
- На странице "Резиновая плитка" - 3 других фильтра
- Один виджет работает на всех страницах!

### Пример 2: Искусственный газон

```
Искусственный газон:
✓ vysota-vorsa (Высота ворса)
✓ naznachenie (Назначение)
✓ cvet (Цвет)
```

## Проверка работы

1. Откройте страницу категории (например `/rezinovye-pokrytiya/`)
2. Должны появиться фильтры, настроенные для этой категории
3. Клик по фильтру переводит на SEO-категорию (если есть) или добавляет query параметр
4. В "Активных фильтрах" показываются выбранные варианты

## Отладка

### Фильтры не показываются?

1. Проверьте, добавлена ли категория в **Filter Attributes**
2. Проверьте, отмечены ли галочки атрибутов
3. Проверьте режим отображения (должен быть "фильтры" или "оба")
4. Убедитесь, что виджет **"SEO Universal Filters"** добавлен в сайдбар

### Показываются не те атрибуты?

1. Откройте **SEO Filter → Filter Attributes**
2. Найдите нужную категорию
3. Измените список атрибутов
4. Сохраните настройки

### Нужно удалить категорию?

1. **SEO Filter → Filter Attributes**
2. Найдите категорию
3. Нажмите кнопку "Удалить" (иконка корзины)

---

## Разница между виджетами

| Виджет | Описание | Когда использовать |
|--------|----------|-------------------|
| **SEO Universal Filters** | Автоматически показывает все фильтры для категории | Всегда (рекомендуется) |
| **SEO Filter Widget** | Показывает один конкретный атрибут | Когда нужен контроль над порядком |
| **SEO Active Filters** | Показывает активные фильтры | Всегда добавляйте вместе с фильтрами |

## Рекомендуемая настройка

```php
// В functions.php или через виджеты

// 1. Универсальные фильтры
the_widget('WSF_Universal_Filters_Widget', [
    'title' => 'Фильтры',
    'display_type' => 'list'
]);

// 2. Активные фильтры
the_widget('WSF_Active_Filters_Widget', [
    'title' => 'Выбранные фильтры'
]);
```

Всё! Фильтры будут работать на всех категориях автоматически.
