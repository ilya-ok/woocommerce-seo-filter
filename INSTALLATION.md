# Инструкция по установке и настройке

## Шаг 1: Установка плагина

1. Создайте папку `woo-seo-filter` в `/wp-content/plugins/`
2. Скопируйте все файлы в созданную папку, соблюдая структуру:

```
wp-content/plugins/woo-seo-filter/
├── woo-seo-filter.php
├── includes/
│   ├── class-autoloader.php
│   ├── class-wsf-plugin.php
│   ├── class-wsf-installer.php
│   ├── class-wsf-admin.php
│   ├── class-wsf-binding-manager.php
│   ├── class-wsf-filter-manager.php
│   ├── class-wsf-redirect-handler.php
│   ├── class-wsf-widget-manager.php
│   ├── class-wsf-filter-widget.php
│   ├── class-wsf-active-filters-widget.php
│   ├── class-wsf-category-helper.php
│   ├── class-wsf-query-handler.php
│   └── class-wsf-shortcodes.php
├── templates/admin/
│   ├── settings.php
│   ├── binding-item.php
│   └── attributes.php
└── assets/
    ├── css/
    │   ├── admin.css
    │   └── frontend.css
    └── js/
        ├── admin.js
        └── frontend.js
```

3. Активируйте плагин в админке WordPress (Плагины → Установленные)

## Шаг 2: Создание атрибутов WooCommerce

Перед настройкой плагина убедитесь, что у вас созданы необходимые атрибуты:

1. Перейдите в WooCommerce → Атрибуты
2. Создайте атрибуты, например:
   - `vysota-vorsa` (Высота ворса)
   - `material` (Материал)
   - `tip` (Тип)
   - `naznachenie` (Назначение)
3. Добавьте значения для каждого атрибута

## Шаг 3: Создание категорий

1. Перейдите в Товары → Категории
2. Создайте основные категории, например:
   - Искусственный газон (slug: `iskusstvennyj-gazon`)
   - Спортивные покрытия (slug: `sportivnye-pokrytiya`)
3. Создайте подкатегории для фильтров:
   - 10 мм газон (slug: `10-mm-gazon`)
   - 15 мм газон (slug: `15-mm-gazon`)
   - Резиновые покрытия (slug: `rezinovye-pokrytiya`)

**ВАЖНО:** Используйте уникальные slug для всех категорий!

## Шаг 4: Настройка связок (Filter Bindings)

1. Перейдите в SEO Filter → Filter Bindings
2. Нажмите "Добавить связку"
3. Заполните данные:
   - **Категория**: выберите категорию для связки (например, `10-mm-gazon`)
   - **Родительская категория**: выберите родителя (например, `iskusstvennyj-gazon`) или "Каталог WooCommerce"
   - **Атрибуты**: нажмите "Добавить атрибут"
     - Выберите атрибут (например, `vysota-vorsa`)
     - Выберите значение (например, `10-mm`)
4. Повторите для всех нужных категорий
5. Используйте Drag & Drop для изменения порядка
6. Нажмите "Сохранить связки"

### Пример связки:
```
Категория: 10-mm-gazon
Родитель: iskusstvennyj-gazon
Атрибуты:
  - vysota-vorsa = 10-mm
```

### Пример многоуровневой связки:
```
Категория: rezinovaya-plitka-dlya-dachi
Родитель: rezinovaya-plitka
Атрибуты:
  - material = rezinovoe
  - tip = plitka
  - naznachenie = dlya-dachi
```

## Шаг 5: Настройка атрибутов категорий

1. Перейдите в SEO Filter → Filter Attributes
2. Для каждой категории выберите:
   - **Источник атрибутов**: 
     - "Из родительской категории" - использовать атрибуты родителя
     - "Из своей категории" - использовать собственные атрибуты
   - **Отображение**:
     - "Показывать фильтры" - только фильтры
     - "Показывать подкатегории" - только подкатегории
     - "Показывать оба" - и фильтры, и подкатегории
3. Нажмите "Сохранить настройки"

## Шаг 6: Добавление виджетов

### Вариант 1: Через виджеты WordPress

1. Перейдите в Внешний вид → Виджеты
2. Найдите виджет "SEO Filter Widget"
3. Перетащите в нужную область (обычно в сайдбар магазина)
4. Настройте:
   - Заголовок: например, "Высота ворса"
   - Атрибут: выберите `vysota-vorsa`
   - Тип отображения: список или выпадающий список
5. Добавьте виджет "SEO Active Filters" для отображения активных фильтров

### Вариант 2: Через шорткоды

В любом месте темы или в редакторе блоков:

```php
// Фильтр по высоте ворса
<?php echo do_shortcode('[wsf_filters attribute="vysota-vorsa" title="Высота ворса" type="list"]'); ?>

// Активные фильтры
<?php echo do_shortcode('[wsf_active_filters title="Активные фильтры"]'); ?>
```

## Шаг 7: Проверка работы

1. Перейдите на страницу категории (например, `/iskusstvennyj-gazon/`)
2. Кликните на фильтр
3. Проверьте, что URL изменился на SEO-категорию (например, `/iskusstvennyj-gazon/10-mm-gazon/`)
4. Убедитесь, что товары отфильтрованы правильно

## Решение проблем

### Фильтры не отображаются
- Проверьте, что категория добавлена в связках
- Убедитесь, что атрибуты созданы в WooCommerce
- Очистите кэш WordPress

### Редирект не работает
- Проверьте настройки постоянных ссылок (Настройки → Постоянные ссылки)
- Пересохраните постоянные ссылки
- Убедитесь, что связка создана для нужных атрибутов

### Товары не фильтруются
- Проверьте, что у товаров назначены нужные атрибуты
- Убедитесь, что значения атрибутов совпадают со slug в связках
- Проверьте query параметры в URL

## Дополнительно

### Синхронизация на мультисайт
Плагин использует slug вместо ID, что позволяет экспортировать настройки:

1. Экспортируйте опции `wsf_bindings`, `wsf_category_attributes`, `wsf_category_display_settings`
2. Убедитесь, что slug категорий совпадают на всех сайтах
3. Импортируйте настройки на другие сайты мультисайта

### Отладка
Включите WP_DEBUG для просмотра ошибок:
```php
define('WP_DEBUG', true);
define('WP_DEBUG_LOG', true);
```

Логи сохраняются в `/wp-content/debug.log`
