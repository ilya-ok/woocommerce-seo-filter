# Ключевые концепции WooCommerce SEO Filter

## 1. Связки (Bindings)

**Связка** = соответствие между категорией и набором атрибутов.

```php
[
    'category_slug' => 'rezinovaya-plitka',
    'parent_slug' => 'rezinovoe',
    'attributes' => [
        ['attribute' => 'material', 'value' => 'rezinovoe'],
        ['attribute' => 'tip', 'value' => 'rezinovoe-plitka']
    ],
    'order' => 1
]
```

**Хранятся в:** `wsf_bindings` (WordPress option)

## 2. Настройки категорий

- **wsf_category_attributes** — источник атрибутов ('parent' или 'own')
- **wsf_category_display_settings** — режим отображения ('filters', 'subcategories', 'both')
- **wsf_category_available_attributes** — доступные атрибуты для фильтрации (массив slug атрибутов)
- **wsf_category_badge_attributes** — атрибуты для виджета меток (массив slug атрибутов)

## 3. Логика работы фильтров

**При клике на фильтр:**
1. Собираются ВСЕ активные атрибуты (из категории + из query)
2. Добавляется новый выбранный атрибут
3. Поиск связки с точным совпадением
4. Если не найдено — поиск частичного совпадения
5. Остальные атрибуты добавляются как query параметры

**При снятии фильтра:**
1. Удаляется атрибут из активных
2. Поиск связки для оставшихся атрибутов
3. Переход на найденную категорию + query для остатка

**Проверка активности:**
- Фильтр активен, если его значение есть в атрибутах текущей категории ИЛИ в query параметрах

## 4. Виджеты

**WSF_Universal_Filters_Widget** — основной виджет
- Автоматически показывает все настроенные атрибуты для категории
- Один виджет для всех категорий
- Используется для основных фильтров

**WSF_Badges_Widget** — виджет меток
- Компактное отображение (inline/vertical)
- Для второстепенных или дополнительных фильтров
- Отдельная настройка атрибутов
- Автоматическое сворачивание в одну строку с кнопками "Показать еще"/"Свернуть"
- Кнопки отображаются только при переполнении (если метки не помещаются в одну строку)
- JavaScript определяет overflow и управляет видимостью кнопок

**WSF_Active_Filters_Widget** — активные фильтры
- Показывает выбранные фильтры
- Каждый фильтр — кликабельная ссылка для удаления
- Кнопка "Сбросить все"

**WSF_Filter_Widget** — одиночный фильтр (legacy)
- Для случаев, когда нужен контроль над порядком
- Один атрибут на виджет

## 5. Синхронизация мультисайта

**WSF_Multisite_Sync** — синхронизация настроек между сайтами

**Функционал:**
- Экспорт настроек из любого сайта в JSON
- Синхронизация настроек с одного сайта на другие
- Массовая синхронизация на все сайты сети
- Страница управления в Network Admin (Settings → SEO Filter Sync)

**Что синхронизируется:**
- Filter Bindings (связки категорий и атрибутов)
- Category Attributes Settings (источник атрибутов — parent/own)
- Category Display Settings (режим отображения)
- Available Attributes (доступные атрибуты для фильтров)
- Badge Attributes (атрибуты для виджета меток)

**Важно:**
- Синхронизация работает по slug категорий и атрибутов
- На всех сайтах должны существовать одинаковые категории и атрибуты
- После синхронизации автоматически очищается кэш на целевых сайтах

**Использование:**
1. Перейти в Network Admin → Settings → SEO Filter Sync
2. Выбрать сайт-источник (откуда копировать)
3. Выбрать сайты-получатели (куда копировать)
4. Нажать "Синхронизировать выбранные" или "Синхронизировать на все сайты"
